# Installation


```sh
pip install 'xpwebapi @ git+https://github.com/devleaks/xplane-webapi.git'
```

For development, add option `dev`


```sh
pip install 'xpwebapi[dev] @ git+https://github.com/devleaks/xplane-webapi.git'
```
